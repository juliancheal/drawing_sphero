## Getting up and running

Run the following in terminal

	git clone https://github.com/byrichardpowell/12devsNodeJsDraw.git
	cd 12devsNodeJsDraw
	npm install
	node app.js

Now browse to http://127.0.0.1:3000/.  Happy drawing!
